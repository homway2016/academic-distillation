# 评审标准框架 — 通用七维 + 分类型标准

## 1. 通用七维（所有论文类型适用）

### Dimension 1: Originality（原创性）— Weight 15%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 提出全新理论/方法/证据，可能改变领域方向 |
| Strong | 4 | 有清晰的新见解或新颖组合，填补特定研究空白 |
| Adequate | 3 | 增量贡献，对现有知识的合理延伸 |
| Weak | 2 | 与现有文献高度重叠，新贡献不清晰 |
| None | 1 | 基本上是已知内容的重复 |

### Dimension 2: Methodological Rigor（方法严谨性）— Weight 25%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 研究设计 impeccable，方法创新且执行 flawless |
| Strong | 4 | 设计合理，方法适当，执行有轻微改进空间 |
| Adequate | 3 | 方法基本可接受，但存在设计或执行局限 |
| Weak | 2 | 方法有重大缺陷，影响结论可信度 |
| Unacceptable | 1 | 方法根本不适合回答研究问题，或包含严重错误 |

### Dimension 3: Evidence Sufficiency（证据充分性）— Weight 20%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 丰富、多元、有说服力的证据，超出预期 |
| Strong | 4 | 证据足以支撑所有主要论点 |
| Adequate | 3 | 大多数论点有证据支撑，少数需要补充 |
| Weak | 2 | 关键论点缺乏充分证据 |
| Unacceptable | 1 | 论点与证据严重脱节 |

### Dimension 4: Argument Coherence（论证连贯性）— Weight 15%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 论点清晰，逻辑严密，结构优雅 |
| Strong | 4 | 论证流畅，偶有轻微逻辑跳跃 |
| Adequate | 3 | 基本连贯，但部分段间连接不清晰 |
| Weak | 2 | 多处逻辑断裂，读者难以跟随 |
| Unacceptable | 1 | 论证混乱，核心主张无法识别 |

### Dimension 5: Writing Quality（写作质量）— Weight 10%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 精确流畅的学术英语/中文，学术写作典范 |
| Strong | 4 | 语言清晰，偶有小瑕疵但不影响理解 |
| Adequate | 3 | 总体可读，有一些语法或用词问题 |
| Weak | 2 | 频繁的语言问题影响理解 |
| Unacceptable | 1 | 语言质量未达到可评审标准 |

### Dimension 6: Literature Integration（文献整合）— Weight 10%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 全面、当代、批判性整合，研究空白论证 compelling |
| Strong | 4 | 涵盖主要文献，整合与定位良好 |
| Adequate | 3 | 基本覆盖，但有遗漏或整合不足 |
| Weak | 2 | 文献过时、不完整或仅枚举 |
| Unacceptable | 1 | 文献综述严重不足或与主题无关 |

### Dimension 7: Significance & Impact（意义与影响）— Weight 5%

| 等级 | 分数 | 描述 |
|------|------|------|
| Outstanding | 5 | 可能改变政策、实践或理论方向 |
| Strong | 4 | 对特定领域或实践有清晰影响 |
| Adequate | 3 | 有一定的学术或实践价值 |
| Weak | 2 | 影响范围有限，主要是学术兴趣 |
| Marginal | 1 | 难以看出研究意义 |

## 2. 分类型评审标准

### 2.1 实证研究（Empirical Research）

额外关注：
- 研究假设清晰度：假设是否可检验、与理论一致
- 变量操作化定义：自变量/因变量/控制变量定义是否精确
- 内部效度：混淆变量是否得到控制
- 外部效度：结果的可推广性
- 统计报告完整性：效应量、置信区间、假设检验
- 结论保守性：结论是否超出数据支持范围

### 2.2 理论/概念论文（Theoretical/Conceptual）

额外关注：
- 概念定义精确性：核心概念是否清晰界定
- 论证逻辑结构：前提→推论→结论链是否完整
- 反论点处理：是否考虑并回应可能的反对观点
- 理论新颖性：是否真正推进理论发展
- 可检验性：理论是否能生成可检验的命题

### 2.3 文献综述 / 元分析（Literature Review / Meta-analysis）

额外关注：
- 检索策略：是否全面且可复现（PRISMA 合规）
- 纳入/排除标准：标准是否清晰、合理、一致应用
- 偏倚风险评估：纳入研究的偏倚风险是否被评估
- 异质性处理：统计与概念异质性是否得到适当处理
- 综合方法：超越简单计票，实现批判性综合
- 发表偏倚：是否评估并讨论发表偏倚

### 2.4 案例研究（Case Study）

额外关注：
- 案例选择理由：为何选择此案例？它代表什幺？
- 理论抽样 vs 便利抽样：案例选择是否有理论基础
- 三角验证：是否使用多种数据来源
- 情境描述厚度：厚描述是否充分
- 分析可迁移性：分析结果是否可迁移到其他情境
- 研究者反身性：研究者与案例的关系是否被反思

### 2.5 政策分析 / 政策简报（Policy Analysis / Brief）

额外关注：
- 政策问题定义：问题是否清晰定义且有证据支撑
- 利益相关者分析：关键利益相关者是否被识别
- 政策选项分析：是否提出并比较多种选项
- 可行性评估：政策建议是否实际可行
- 证据质量：政策建议是否基于可靠证据
- 非预期后果：是否考虑政策的非预期影响

## 3. 评审者应避免的偏见

| 偏见 | 描述 | 如何避免 |
|------|------|---------|
| **过度批评** | 夸大 minor issues，忽略整体贡献 | 先肯定优点，再指出问题；区分 major/minor |
| **确认偏误** | 只找支持自己预设观点的证据 | 刻意寻找论文优点和自己观点的反例 |
| **偏好投射** | 要求作者用"我的方法"而非评估"作者的方法" | 问"这个方法能回答问题吗"而非"我会怎幺做" |
| **范式偏见** | 用定量标准评判定性研究（或反之） | 使用与论文研究范式匹配的评估标准 |
| **声望偏见** | 因作者机构或过往成就放松标准 | 专注于论文本身的质量 |
| **新颖性偏见** | 只重视新颖研究，低估重复研究 | 承认重复研究在科学中的重要角色 |
| **长度偏见** | 长论文=好论文，短论文=粗糙 | 评估内容密度，而非页数 |
| **语言歧视** | 因非母语语言缺陷低估研究质量 | 区分"语言需要润色"与"研究质量差" |

## 4. 建设性反馈原则

1. **具体，不模糊**："第3节第2段的因果推断缺少控制变量" 优于 "方法论有问题"
2. **问题 + 理由 + 建议**：每条批评都应包含"什幺"、"为什幺"、"怎幺改"
3. **区分必需与建议**：哪些修改是强制的，哪些是"锦上添花"
4. **承认不确定性**："我不确定此分析是否考虑了 X" 比 "作者忽略了 X" 更准确
5. **尊重作者**：即使论文质量差，作者也投入了时间和精力

## 5. 分数聚合与决策映射

```
总分 = Originality×0.15 + Methodology×0.25 + Evidence×0.20
     + Coherence×0.15 + Writing×0.10 + Literature×0.10 + Significance×0.05
```

| 加权总分 | 推荐决策 |
|---------|---------|
| 4.5–5.0 | Accept |
| 3.5–4.4 | Minor Revision |
| 2.5–3.4 | Major Revision |
| 1.5–2.4 | Reject (Resubmit encouraged) |
| 1.0–1.4 | Reject |

**重要提醒**：
- 若任何单一维度特别低（如方法论=1），即使总分尚可也可能 Reject
- 评审意见的具体内容比数字更重要
- 需考虑期刊特殊背景（特刊、领域发展需要等）
