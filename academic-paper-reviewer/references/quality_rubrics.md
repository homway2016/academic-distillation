# 质量量表 — 0–100 评分细则

## 决策映射

| 加权平均分 | 决策 |
|-----------|------|
| ≥ 80 | Accept |
| 65–79 | Minor Revision |
| 50–64 | Major Revision |
| < 50 | Reject |

---

## Dimension 1: Originality（原创性）— Weight 20%

| 分数段 | 描述符 | 行为指标 |
|--------|--------|---------|
| 90–100 | Exceptional | 新理论框架 + 实证支撑；开辟全新研究方向；影响横跨 3+ 领域；无先前工作触及此精确问题 |
| 75–89 | Strong | 新方法或现有理论的新情境应用；超越增量延伸的清晰贡献；影响 2+ 领域 |
| 60–74 | Adequate | 用新数据/人群/情境扩展现有框架；贡献清晰但增量；单领域影响 |
| 45–59 | Weak | 以微小变化重复现有研究；贡献边缘；"so what?" 未令人信服地回答 |
| < 45 | Insufficient | 无可见原创贡献；无正当理由复制现有工作；纯描述无分析洞察 |

## Dimension 2: Methodological Rigor（方法严谨性）— Weight 25%

| 分数段 | 描述符 | 行为指标 |
|--------|--------|---------|
| 90–100 | Exceptional | 研究设计与 RQ 完美对齐；所有效度威胁被处理；适当统计方法 + 功效分析；透明报告（全部 EQUATOR 条目）；可复现 |
| 75–89 | Strong | 设计合理有小缺陷；多数效度威胁被处理；适当方法有小报告遗漏；大体可复现 |
| 60–74 | Adequate | 设计可接受但有效度担忧；方法适当但缺乏论证；部分报告遗漏（缺效应量、CI） |
| 45–59 | Weak | 设计有重大缺陷；方法选择 questionable；多处报告遗漏；复现性可疑 |
| < 45 | Insufficient | 根本性设计缺陷使结论无效；方法不当；结果不可信 |

## Dimension 3: Evidence Sufficiency（证据充分性）— Weight 25%

| 分数段 | 描述符 | 行为指标 |
|--------|--------|---------|
| 90–100 | Exceptional | >40 来源，80%+ 同行评审，多方法三角验证，一手+二手数据，所有主张均有支撑，反证被承认 |
| 75–89 | Strong | 25–40 来源，70%+ 同行评审，主要主张有充分证据，部分三角验证 |
| 60–74 | Adequate | 15–25 来源，60%+ 同行评审，关键主张有支撑但部分有缺口，三角验证有限 |
| 45–59 | Weak | <15 来源 或 <50% 同行评审，若干无支撑主张，无三角验证 |
| < 45 | Insufficient | 严重欠来源，主要主张无支撑，重度依赖灰色文献或轶事证据 |

## Dimension 4: Argument Coherence（论证连贯性）— Weight 15%

| 分数段 | 描述符 | 行为指标 |
|--------|--------|---------|
| 90–100 | Exceptional | 从问题→空白→RQ→方法→发现→启示的 Crystal-clear 逻辑流；每节构建于前者；无逻辑跳跃；反论点被预判 |
| 75–89 | Strong | 清晰逻辑流有小缺口；多数过渡处理良好；论证总体有说服力 |
| 60–74 | Adequate | 主论证可见但部分章节感觉脱节；偶有逻辑跳跃；结论大体跟随证据 |
| 45–59 | Weak | 论证结构不清晰；显着逻辑缺口；结论超出证据范围；读者必须推断连接 |
| < 45 | Insufficient | 无连贯论证；章节看似无关；结论不从证据推出；循环论证 |

## Dimension 5: Writing Quality（写作质量）— Weight 15%

| 分数段 | 描述符 | 行为指标 |
|--------|--------|---------|
| 90–100 | Exceptional | 专业学术散文；精确术语；优秀段落结构；零语法错误；全程适当语域 |
| 75–89 | Strong | 良好学术写作；轻微风格不一致；少数语法问题；术语大体精确 |
| 60–74 | Adequate | 可接受写作但有改进空间；部分冗长段落；偶尔术语不精确；一些语法问题 |
| 45–59 | Weak | 低于期刊标准；频繁冗长/不清晰段落；术语不一致；多处语法问题 |
| < 45 | Insufficient | 不可接受的写作质量；不可理解的段落；严重语法问题；不适合同行评审 |

## 可选维度（报告用，不计入总分）

### Literature Integration（R2 领域专家重点）

| 分数段 | 描述符 |
|--------|--------|
| 90–100 | 全面覆盖经典+近期工作；识别理论谱系；精确定位论文于学术对话中 |
| 75–89 | 良好覆盖；多数关键作品被引用；文献中定位合理 |
| 60–74 | 可接受但有覆盖缺口；部分重要作品缺失；定位 somewhat vague |
| < 60 | 显着文献缺口；关键作品缺失；定位差 |

### Significance & Impact（R3 跨学科视角重点）

| 分数段 | 描述符 |
|--------|--------|
| 90–100 | 对政策/实践 AND 理论有清晰实践意义；解决紧迫现实问题；可能影响领域方向 |
| 75–89 | 良好实践 OR 理论意义；解决相关问题；中等影响潜力 |
| 60–74 | 有一些意义但范围狭窄；相关性清晰但影响有限 |
| < 60 | 最小实践或理论意义；不清楚为什幺重要 |

---

## 聚合公式

```
Final Score = (Originality × 0.20) + (Methodology × 0.25) + (Evidence × 0.25)
            + (Coherence × 0.15) + (Writing × 0.15)
```

可选维度单独报告，纳入编辑综合叙事，但不改变数值总分。

## 校准注意事项

- 分数应相对于**目标期刊标准**反映论文质量
- Nature 的 "75" 与区域性期刊的 "75" 不等价
- 不确定时，倾向于范围中间值
- 评审者应明确说明哪个范围描述符最匹配，然后在该范围内微调
- 若两个维度冲突（如优秀方法论但写作差），**不要平均拉低**——如实报告两个分数
