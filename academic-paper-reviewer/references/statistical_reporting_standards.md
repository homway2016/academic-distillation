# 统计报告标准 — 统计审查清单 + APA 7.0 + 红旗检测

## 1. 通用统计报告检查清单

### 1.1 描述统计

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| 均值 (*M*) | 所有连续变量必须报告 | 仅报告总体，未按组报告 |
| 标准差 (*SD*) | 必须与均值成对出现 | 误用标准误 (*SE*) 替代 SD |
| 样本量 (*N* / *n*) | 必须报告总体和分组样本量 | 分析过程中的样本流失未解释 |
| 范围 | 报告 Min–Max 或四分位距 | 完全缺失，无法判断分布 |
| 分类变量分布 | 报告频数 (*f*) 和百分比 (%) | 仅报告百分比，缺失原始频数 |

### 1.2 效应量

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| 报告义务 | **所有统计检验必须附带效应量** — APA 7.0 强制要求 | 仅报告 *p* 值，无效应量 |
| 选择适当指标 | 按分析方法选择对应效应量指标 | 使用不当的效应量指标 |
| 解释 | 必须提供 Cohen 常规基准或领域基准 | 报告数字但未解释大小 |

**常用效应量指标速查：**

| 分析方法 | 效应量指标 | 小/中/大 (Cohen) |
|----------|-----------|-----------------|
| *t*-test | Cohen's *d* | 0.2 / 0.5 / 0.8 |
| ANOVA | *eta*-squared | .01 / .06 / .14 |
| Correlation | *r* | .10 / .30 / .50 |
| Regression | *R*-squared, *f*-squared | *f*²: .02 / .15 / .35 |
| Chi-square | Cramer's *V*, *phi* | *V*: .10 / .30 / .50 (*df*=1) |
| Odds Ratio | OR | 1.5 / 2.5 / 4.3 |

### 1.3 置信区间

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| CI 报告 | 所有效应量和关键估计**应**报告 95% CI | CI 完全缺失 |
| 格式 | 95% CI [下限, 上限] | 使用圆括号或不一致格式 |
| 解释 | 描述 CI 的实质意义，不止统计意义 | 仅检查 CI 是否包含零 |

### 1.4 统计显着性

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| *p* 值格式 | 报告精确 *p* 值（如 *p* = .032） | 仅报告 *p* < .05 |
| *p* < .001 | *p* 很小时可报告 *p* < .001 | 报告 *p* = .000（原始软件输出） |
| Alpha 水平 | 先验声明 alpha 水平 | 未声明 alpha = .05 还是其他 |
| 多重比较 | 使用 Bonferroni、Holm、FDR 校正 | 多重比较无任何校正 |
| 非显着结果 | 必须完整报告；不可隐藏 | 选择性仅报告显着结果 |

### 1.5 统计功效

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| 先验功效分析 | 声明目标功效（通常 ≥ .80）、假设效应量、alpha、所需样本量 | 功效分析完全缺失 |
| 效应量来源 | 基于先前研究、试点研究或理论预期 | 无解释地使用 Cohen 惯例 |
| 工具 | 使用 G*Power、pwr 包等 | 未指定工具 |
| 事后功效 | 对非显着结果报告观察功效 | 未讨论非显着结果的 II 型错误风险 |
| 敏感性分析 | 报告给定 *N* 下的最小可检测效应量 | 未进行敏感性分析 |

### 1.6 缺失数据处理

| 项目 | 标准 | 常见遗漏 |
|------|------|---------|
| 缺失数据报告 | 报告每个变量的缺失数据量和比例 | 未报告缺失数据情况 |
| 缺失机制 | 讨论 MCAR / MAR / MNAR | 未经检验假设 MCAR |
| 处理方法 | 声明使用的方法：listwise / pairwise / MI / FIML | 未声明或仅使用 listwise |
| 敏感性分析 | 比较不同缺失处理方法下的结果稳健性 | 仅用一种方法，未检验敏感性 |

### 1.7 假设检验

| 假设 | 适用分析 | 检验方法 | 常见遗漏 |
|------|---------|---------|---------|
| 正态性 | *t*-test, ANOVA, regression | Shapiro-Wilk / K-S / Q-Q / 偏度峰度 | 完全未检验或仅引用 CLT |
| 方差齐性 | 独立 *t*-test, ANOVA | Levene's test | 未报告或违反时未用替代方法 |
| 线性 | Regression, correlation | 残差图 / 散点图 | 未经检验假设线性 |
| 独立性 | 多数参数检验 | Durbin-Watson / 研究设计说明 | 嵌套数据未处理 |
| 多重共线性 | 多元回归 | VIF, tolerance, 相关矩阵 | VIF 未报告或报告未处理 |
| 残差正态性 / 同方差性 | Regression | 残差图, Breusch-Pagan | 模型拟合后未检查残差 |

## 2. 方法专用检查清单

### 2.1 *t*-test（独立 / 配对样本）

- 报告 *t* 统计量：*t*(df) = X.XX, *p* = .XXX
- 独立 vs 配对：选择正确？配对设计需报告配对逻辑
- 效应量：Cohen's *d*（独立）或 *d*_z（配对）
- 假设检验：正态性（小样本重要）、方差齐性（独立 *t*-test）
- Welch 校正：方差不等时是否使用？
- 方向性：单尾 vs 双尾是否有先验理论基础支持？

### 2.2 ANOVA（单因素 / 因素 / 重复测量）

- 报告 *F* 统计量：*F*(df1, df2) = X.XX, *p* = .XXX
- 效应量：*eta*-squared、partial *eta*-squared 或 *omega*-squared
- 事后比较：主效应显着时是否做 post-hoc（Tukey / Bonferroni / Games-Howell）
- 交互效应：因素设计中是否解释交互？是否检验简单效应？
- 球形假设：重复测量是否报告 Mauchly's test + Greenhouse-Geisser / Huynh-Feldt 校正？
- 不等组大小：组大小差异大时是否使用 Type III SS？

### 2.3 回归分析（线性 / Logistic）

**线性回归**
- 模型摘要：*R*-squared、adjusted *R*-squared、模型 *F* 检验
- 系数表：*B*, *SE*, *beta*, *t*, *p*, 95% CI for *B*
- 多重共线性：VIF（< 5 或 < 10 依领域惯例）、tolerance
- 残差诊断：正态性、同方差性、线性、异常值（Cook's *D*）
- 变量选择：enter vs stepwise 的论证

**Logistic 回归**
- 模型拟合：Hosmer-Lemeshow / chi-squared / -2LL / Nagelkerke *R*-squared
- 系数报告：*B*, *SE*, Wald, OR, 95% CI for OR
- 分类准确率：分类表、敏感度、特异度、AUC/ROC
- 样本量：至少 10–20 events per predictor（EPV 规则）

### 2.4 结构方程模型（SEM）

- 样本量：通常 ≥ 200；或估计参数数量的 5–10 倍
- 模型拟合指标：**必须同时报告多个指标**（至少 4 个）
- CFI / TLI：≥ .95（好）；≥ .90（可接受）
- RMSEA：≤ .06（好）；≤ .08（可接受）；必须报告 90% CI
- SRMR：≤ .08
- 测量模型：SEM 前先做 CFA（两步法）
- 信效度：CR ≥ .70, AVE ≥ .50, 区分效度（Fornell-Larcker / HTMT）

### 2.5 多层线性模型（HLM / MLM）

- 嵌套结构：清楚解释每一层（如学生 → 班级 → 学校）
- ICC：报告组内相关系数；ICC > .05 支持使用 MLM
- 随机效应：报告随机截距和（如适用）随机斜率方差
- 固定效应：报告系数、*SE*、*t* / *z*、*p*、CI
- 中心化：解释使用 grand-mean 还是 group-mean 中心化及原因
- 模型比较：使用 deviance (-2LL)、AIC、BIC 比较嵌套模型

## 3. APA 7th Edition 统计格式速查

### 3.1 数字格式

| 规则 | 正确 | 错误 |
|------|------|------|
| *p* 值无前导零 | *p* = .032 | *p* = 0.032 |
| 可超过 1.0 的统计量有前导零 | *M* = 0.75 | *M* = .75 |
| 不可超过 1.0 的统计量无前导零 | *r* = .45 | *r* = 0.45 |
| 通常 2 位小数 | *M* = 3.45 | *M* = 3.4 |
| *p* 值 2–3 位小数 | *p* = .03 或 *p* = .032 | *p* = .0321 |
| 百分比 0–1 位小数 | 45.2% | 45.2381% |

**不可超过 1.0**（无前导零）：*r*, *R*, *p*-value, Cramer's *V*, *phi*, *eta*-squared, *R*-squared, *beta*

**可超过 1.0**（有前导零）：*M*, *SD*, *B*, Cohen's *d*, *t*, *F*, chi-squared

### 3.2 统计符号斜体规则

| 斜体 | 非斜体 |
|------|--------|
| *M*, *SD*, *SE* | df |
| *N*, *n* | SS, MS |
| *t*, *F*, *p*, *r*, *R*, *z* | OR, CI, VIF |
| *d*, *f*-squared, *eta*-squared | AIC, BIC, CFI, TLI |
| *B*, *beta* | RMSEA, SRMR |
| *chi*-squared | ICC |

### 3.3 统计结果报告格式示例

| 分析方法 | APA 格式示例 |
|----------|-------------|
| 独立样本 *t*-test | *t*(58) = 2.45, *p* = .017, *d* = 0.63, 95% CI [0.12, 1.14] |
| One-way ANOVA | *F*(2, 87) = 4.56, *p* = .013, partial *eta*-squared = .09 |
| Linear regression | *B* = 0.34, *SE* = 0.12, *beta* = .28, *t*(95) = 2.83, *p* = .006, 95% CI [0.10, 0.58] |
| Logistic regression | *B* = 1.24, *SE* = 0.45, Wald = 7.59, *p* = .006, OR = 3.46, 95% CI [1.43, 8.37] |
| SEM fit | chi-squared(52) = 78.34, *p* = .011, CFI = .97, TLI = .96, RMSEA = .045 [.018, .068], SRMR = .038 |

## 4. 统计红旗清单

### 4.1 P-hacking 指标

| 红旗 | 描述 | 严重度 |
|------|------|--------|
| 多个 *p* 集中在 .04–.05 | 结果 *p* 集中在 .04–.05 范围 | HIGH |
| 选择性报告 | 仅报告显着结果，非显着结果消失 | HIGH |
| 模糊分析策略 | 分析策略未先验声明，事后看起来是探索性的 | MEDIUM |
| 意外子群 | 事后子群分解以寻找显着结果 | MEDIUM |
| 灵活样本量 | 无预定义停止规则（序贯检验无校正） | HIGH |
| "排除异常值" | 大量异常值被排除且标准不清 | MEDIUM |

### 4.2 HARKing

| 红旗 | 描述 | 严重度 |
|------|------|--------|
| 假设与结果完美匹配 | 所有假设都被支持，无例外 | MEDIUM |
| 探索性包装为验证性 | 文献综述明显是事后构建的 | HIGH |
| 假设方向改变 | 原本预测正向但结果为负，仍声称"如预期" | HIGH |

### 4.3 其他红旗

| 红旗 | 描述 | 严重度 |
|------|------|--------|
| 无效应量 | 结论仅基于 *p* 值 | HIGH |
| CI 完全缺失 | 无法判断估计精度 | MEDIUM |
| 极宽 CI | CI 从小效应跨到大效应 | MEDIUM |
| 不一致效应量 | 报告的效应量与原始数据计算不一致 | HIGH |
| 无功效分析 | 样本量缺乏先验计算基础 | MEDIUM |
| 多重比较未校正 | 3+ 组比较使用多个 *t*-test | HIGH |
| ANOVA 后无 post-hoc | 主效应显着但声称组差异无 post-hoc | MEDIUM |
| 假设检验完全缺失 | 跳过正态性/方差齐性/线性检验 | MEDIUM |
| 违反未处理 | 报告了违反但原分析仍使用 | HIGH |
| *p* = .000 | 原始统计软件输出，应写 *p* < .001 | LOW |
| df 与 *N* 不一致 | 从自由度推导的 *N* 与报告 *N* 不匹配 | HIGH |
| 因果语言 | 非实验设计（相关/调查）使用因果推断语言 | MEDIUM |

## 5. 推荐评审顺序

```
Step 1: 确认研究问题 → 分析方法对应关系是否合理
Step 2: 检查是否报告假设检验
Step 3: 逐项检查通用清单（1.1–1.6）
Step 4: 查阅方法专用检查清单（Section 2）
Step 5: 扫描红旗清单（Section 4）
Step 6: 验证 APA 格式（Section 3）
```
